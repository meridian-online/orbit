**Category:** environment-mismatch
**Description:** `.orbit/cards/memos/` still exists as an empty directory on disk after the migration. AC verification (`git ls-files .orbit/cards/memos/` returns nothing) passes nonetheless.
**Evidence:**
```
$ test -d .orbit/cards/memos && echo "STILL EXISTS as directory"
STILL EXISTS as directory
$ ls -la .orbit/cards/memos/
(empty)
$ git ls-files .orbit/cards/memos/
(empty)
```
**Recommendation:** `rmdir .orbit/cards/memos` once on the implementer's machine. Not blocking — the directory is invisible to git, won't propagate via push, and a fresh clone will not have it.
